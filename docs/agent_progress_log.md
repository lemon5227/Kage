# Agent Progress Log

本文件是实施台账。每次改动必须可追溯、可复现、可回滚。

## 记录规则
- 每个任务使用唯一 Task ID（M1-*, M2-* ...）。
- 必须记录"为什么这么做"，不能只写"做了什么"。
- 必须附验证命令与指标前后对比。
- 未达到验收标准不得标记完成。

## 任务记录模板（复制使用）

```md
### [Task ID] Mx-y - 标题
- Date:
- Owner:
- Status: pending | in_progress | done | blocked
- Change Summary:
- Files Touched:

- Next Step:
```
