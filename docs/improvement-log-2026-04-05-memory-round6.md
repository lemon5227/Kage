## 二、实施内容

### 2.1 LLM 辅助事实提取器

**新增**: `core/memory_llm_extractor.py`

使用本地 LLM 提取复杂事实，比规则提取更准确：
- 结构化 JSON 输出
- 支持否定检测
- 自动回退到规则提取
- 合并策略：LLM 优先，规则回退

**Prompt 设计**:
```
你是一个记忆提取助手。从对话中提取用户的结构化事实。
规则：只提取重要事实，忽略无意义对话，重要性 1-5，分类 6 种...
输出格式：JSON 数组
```

### 2.2 AgenticLoop 集成 LLM 提取

**修改**: `core/agentic_loop.py`

- 新增 `_llm_extractor` 和 `_use_llm_extraction` 配置
- 提取流程：LLM 优先 → 失败回退规则提取
- 同步调用封装（`_extract_facts_sync`）

### 2.3 Bug 修复

**Prompt 转义**: `EXTRACTION_PROMPT` 中的 JSON `{}` 需要转义为 `{{}}`，否则 `str.format()` 会报错。
